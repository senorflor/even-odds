Instructions

To determine whether a number is even or odd (from 0 to 65535), simply enter it in your browser as the last part of the URL (preceded by a / if necessary to separate it from the domain) at https://even-odds.netlify.com.

If you are running this "project" locally, add a .htm to the end of the number, since you won't have Netlify's routing defaults resolving to htm/html pages for you.

(You may have to run it locally, because this "website" has been deploying on Netlify for 27 minutes.)
